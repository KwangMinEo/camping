# [4인팀 프로젝트] 캠핑사이트

> 자라섬 캠핑장 예약하기 위한 웹사이트와 DB 설계 및 구현

> 이클립스 MVC1으로 구현

> 2020-02-03 ~ 2020-02-14

## 개발환경
* JDK 1.8.0
* Apache Tomcat 9
* Eclipse
* Oracle SQL 11g
* Github
* JSP
* JAVA
* Jquery
* JavaScript
* HTML5
* CSS3
* Bootstrap
* EL

## 주요기능<CRUD>
* 홈 화면
* 캠핑장 소개페이지
* 예약페이지
* 예약확인페이지
* 예약취소하기
* 예약후기
* 공지사항 페이지
